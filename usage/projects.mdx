## Create Project

Create a new project.

<CodeGroup>

```javascript JavaScript
const project = await vartiq.project.create({
  name: "Project name",
  description: "Optional description" // optional
});
```

```go Go
project, err := client.Project.Create(ctx, &vartiq.CreateProjectRequest{
  Name: "Project name",
  Description: "Optional description", // optional
})
```

```python Python
project = vartiq.project.create(
  name="Project name",
  description="Optional description"  # optional
)
```

```php PHP
$project = $vartiq->project->create([
  "name" => "Project name",
  "description" => "Optional description" // optional
]);
```

</CodeGroup>

## Update Project

Update an existing project.

<CodeGroup>

```javascript JavaScript
const project = await vartiq.project.update("PROJECT_ID", {
  name: "New name", // optional
  description: "New description" // optional
});
```

```go Go
project, err := client.Project.Update(ctx, "PROJECT_ID", &vartiq.UpdateProjectRequest{
  Name: "New name", // optional
  Description: "New description", // optional
})
```

```python Python
project = vartiq.project.update("PROJECT_ID",
  name="New name",  # optional
  description="New description"  # optional
)
```

```php PHP
$project = $vartiq->project->update("PROJECT_ID", [
  "name" => "New name", // optional
  "description" => "New description" // optional
]);
```

</CodeGroup>
