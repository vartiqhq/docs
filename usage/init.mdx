## Initialize the SDK

<CodeGroup>

```javascript JavaScript
import { Vartiq } from 'vartiq';

const vartiq = new Vartiq("YOUR_API_KEY");
```

```go Go
import('github.com/vartiqhq/vartiq-go-sdk/vartiq');

vartiq := vartiq.New("YOUR_API_KEY")
```

```python Python
from vartiq import Vartiq

vartiq = Vartiq("YOUR_API_KEY")
```

```php PHP
use Vartiq\Vartiq;

$vartiq = new Vartiq("YOUR_API_KEY");
```

</CodeGroup>
