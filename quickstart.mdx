## Concepts

Vartiq has three main concepts:

- **Apps** represents your customer. Create one app per customer.
- **Webhooks** are the URLs that receive the webhooks. Each app can have multiple webhooks.
- **Messages** are the the data that is sent to the webhooks.

## Initialize the SDK

<CodeGroup>

```javascript JavaScript
import { Vartiq } from 'vartiq';

const vartiq = new Vartiq("YOUR_API_KEY");
```

```go Go

import('github.com/vartiqhq/vartiq-go-sdk/vartiq');

vartiq := vartiq.New("YOUR_API_KEY")

````

```python Python
from vartiq import Vartiq

vartiq = Vartiq("YOUR_API_KEY")
````

```php PHP
use Vartiq\Vartiq;

$vartiq = new Vartiq("YOUR_API_KEY");
```

</CodeGroup>

## Creating an App

<CodeGroup>

```javascript JavaScript
const app = await vartiq.app.create({
  name: "App name", // setting this to the name or id of your customer is recommended
  projectId: "PROJECT_ID",
});
```

```go Go
app, err := client.App.Create(ctx, &vartiq.CreateAppRequest{
	Name:      "App name", // setting this to the name or id of your customer is recommended
	ProjectID: "PROJECT_ID",
})
```

```python Python
app = vartiq.app.create(
  name="App name", // setting this to the name or id of your customer is recommended
  project_id="PROJECT_ID",
)
```

```php PHP
$app = $vartiq->app->create([
  "name" => "App name", // setting this to the name or id of your customer is recommended
  "project_id" => "PROJECT_ID",
]);
```

```bash cURL
curl -X POST https://api.us.vartiq.com/apps \
-H "X-API-KEY: YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"name": "App name", "project_id": "PROJECT_ID"}'
```

</CodeGroup>

## Creating a Webhook

<CodeGroup>

```javascript JavaScript
const webhook = await vartiq.webhook.create({
  url: "https://example.com",
  appId: "APP_ID",
  customHeaders: [{ key: "x-app", value: "x-value" }], // optional
});
```

```go Go
webhookResp, err := client.Webhook.Create(ctx, &vartiq.CreateWebhookRequest{
	Name:   "Webhook",
	URL:    "https://your-webhook-url.com",
	AppID:  "APP_ID",
	CustomHeaders: []vartiq.Header{{Key: "x-app", Value: "x-value"}}, // optional
})
```

```python Python
webhook = vartiq.webhook.create(
  name="Webhook",
  url="https://your-webhook-url.com",
  app_id="APP_ID",
  custom_headers=[{"key": "x-app", "value": "x-value"}]  # optional
)
```

```php PHP
$webhook = $vartiq->webhook->create([
  'name' => 'Webhook',
  'url' => 'https://your-webhook-url.com',
  'appId' => 'APP_ID',
  'customHeaders' => [['key' => 'x-app', 'value' => 'x-value']] // optional
]);
```

```bash cURL
curl -X POST https://api.us.vartiq.com/webhooks \
-H "X-API-KEY: YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"name": "Webhook", "url": "https://your-webhook-url.com", "app_id": "APP_ID", "custom_headers": [{"key": "x-app", "value": "x-value"}]}'
```

</CodeGroup>

## Sending a Message

Sending a message to a app(your customer) will send the message to all the webhooks associated with the app.

<CodeGroup>

```javascript JavaScript
const message = await vartiq.webhook.message.create("APP_ID", { 
    hello: "world" 
});
```

```go Go
// Create a webhook message
message, err := client.WebhookMessage.Create(ctx, "APP_ID", map[string]interface{}{
	"hello": "world",
})
```

```python Python
message = vartiq.webhook.message.create("APP_ID", {"hello": "world"})
```

```php PHP
$message = $vartiq->webhook->message->create("APP_ID", [
  'hello' => 'world'
]);
```

```bash cURL
curl -X POST https://api.us.vartiq.com/webhook-messages \
-H "X-API-KEY: YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"app_id": "APP_ID", "payload": {"hello": "world"}}'
```

</CodeGroup>
